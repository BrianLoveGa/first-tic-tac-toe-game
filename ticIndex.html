So you wanna play Tic Tac Toe ?

You found the right spot

Learn to tic-tac-toe

One must simply click in the grid to make a move.

An X move will return a red square. An O move will result in a blue square

The object is to get three in a row, three in a colum, or three diagonally to win

If all squares are clicked on (filled in red or blue) and there is not 3 in any way (row, column, diagonal) Then the game is a scratch - reset to play again
